# Secure Business Network

## Features
- VLAN 10 (Administration)
- VLAN 20 (IT)
- VLAN 99 (Management)
- VLAN 999 (Unused Ports)
- Router-on-a-Stick
- DHCP
- Port Security
- Trunk Restrictions
- Native VLAN Configuration

## Topology
1 Router
2 Switches
8 PCs

## Troubleshooting
A DHCP pool configuration error caused VLAN 10 hosts to receive incorrect addressing information. The issue was identified by assigning static IP addresses and comparing connectivity results.
